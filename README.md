# Raj Villa Website

A premium static real estate website for Raj Villa by Saurabh Desai.

Public contact number: +91 97275 04111.

## Preview

Run a local server from this folder:

```powershell
python -m http.server 8050
```

Then open:

```text
http://127.0.0.1:8050/
```

The 3D model uses browser module loading, so preview it from a local server instead of opening `index.html` directly.

## Included

- `index.html` - Raj Villa launch page content
- `styles.css` - responsive premium UI system
- `script.js` - mobile navigation, booking form feedback, video behavior, and GLB scroll reveal
- `assets/raj-villa-house.glb` - supplied 3D model
- `assets/raj-villa-tour.mp4` - supplied walkthrough video
- `assets/raj-villa-*.jpeg` / `.png` - supplied project renders
